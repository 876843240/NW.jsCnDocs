NW.js 0.13.0 参考文档

NW.js 让您可以使用WEB(html5、css3、javascript)技术，包括Node.js极其所有modules来开发桌面应用的一种新方式。她的前身是 "node-webkit" 项目。

该文档包含三个主要组成部分:

  • 用户手册 - 当您仅使用NW.js开发应用时参考该部分文档
  • 开发者手册 - 当您想扩展NW.js时候可以参考该部分文档
  • 参考资料 - NW.js API 文档

该项目的所有源码和文档都在GIT仓库中，欢迎您来Pull Requests.
